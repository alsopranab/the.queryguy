# Variables & Data Types

## What it is
Variables store data values used in analysis.

## Common Data Types
- int → whole numbers (e.g., `10`)
- float → decimal numbers (e.g., `10.5`)
- str → text (e.g., `"Sales"`)
- bool → True / False

## Example
`x = 10`  
`price = 99.5`  
`name = "Revenue"`  
`is_active = True`

## Interview Tip
Python is dynamically typed, type is inferred at runtime.
