# IF

## What it does
Checks a condition and returns one value if TRUE and another if FALSE.

## Syntax
`=IF(logical_test, value_if_true, value_if_false)`

## Parameters
- `logical_test` → condition to evaluate
- `value_if_true` → result if condition is TRUE
- `value_if_false` → result if condition is FALSE

## Example
`=IF(A2>=50000, "High Value", "Low Value")`

## Business Use
Customer or order segmentation.

## Interview Note
IF is the foundation of all Excel logic.
