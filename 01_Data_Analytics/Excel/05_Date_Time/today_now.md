# TODAY & NOW

## What they do
Return the current date or current date + time.

## Syntax
`=TODAY()`  
`=NOW()`

## Difference
- `TODAY()` → returns only date  
- `NOW()` → returns date and time  

## Example
`=TODAY()`  
`=NOW()`

## Business Use
Dynamic reports that auto-update daily.

## Interview Note
Both are volatile functions (recalculate on every refresh).
