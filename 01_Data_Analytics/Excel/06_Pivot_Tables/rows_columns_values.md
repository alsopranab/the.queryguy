# Rows, Columns, Values

## Rows
Used for grouping data  
Example: `Region`, `Product`, `Customer`

## Columns
Used for category split  
Example: `Month`, `Year`, `Status`

## Values
Used for calculations  
Common metrics:
- Sum
- Count
- Average
- Max / Min

## Interview Tip
Always explain **why** you placed a field in Rows or Values.
