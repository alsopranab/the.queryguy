# Pivot Tables – Basics

## What it is
A Pivot Table is used to **summarize, analyze, and aggregate large datasets** quickly.

## How to create
- Select data range
- Insert → Pivot Table
- Choose Rows, Columns, Values, Filters

## Typical Use
Summarizing sales, revenue, headcount, performance metrics.

## Interview Line
Pivot Tables are used to quickly convert raw data into insights.
