# Basic Excel Interview Questions

## Q1: Difference between COUNT, COUNTA, COUNTBLANK
- `COUNT(range)` → counts numbers only
- `COUNTA(range)` → counts non-empty cells
- `COUNTBLANK(range)` → counts empty cells

## Q2: Difference between relative and absolute reference
- Relative → `A1`
- Absolute → `$A$1`
- Mixed → `$A1` or `A$1`

## Q3: How to quickly sum a column
Use `=SUM(A:A)` or shortcut Alt + `=`
