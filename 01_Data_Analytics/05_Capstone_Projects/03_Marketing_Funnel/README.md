# Marketing Funnel Analytics Capstone Project

## Business Context
The company runs digital marketing campaigns.

Management notices:
- High website traffic
- Low final conversions

They want to understand **where users drop off**.

## Business Questions
1. How many users enter each funnel stage?
2. Where is the highest drop-off?
3. What is the overall conversion rate?
4. Which channels perform best?
5. How can conversions be improved?

## Tools Used
- SQL → funnel stage aggregation
- Python → conversion analysis
- Statistics → drop-off interpretation
- Power BI → funnel visualization
