# Sales Analytics Capstone Project

## Business Context
A retail company sells products across multiple regions.

Management has noticed:
- Revenue is increasing
- Profit is inconsistent
- Some regions perform better than others

They want data-driven answers.

## Business Questions
1. How is sales performing over time?
2. Which regions and products drive revenue?
3. Why is profit volatile despite revenue growth?
4. What actions should management take?

## Tools Used
- SQL → data extraction
- Python → cleaning & analysis
- Excel → quick checks
- Power BI → dashboard
- Statistics → variance & trends
